# Scraping
Install :

Node.Js

Instructions

Setup

Clone this repo:
git clone https://github.com/Suhailali23/Scraping

Run this command:

npm install

Running the code

1) Execute this command in your terminal:

node app.js

2) Open the app in your browser:

http://localhost:8080

3) Enter a URL in the text box and then click: "GO"
